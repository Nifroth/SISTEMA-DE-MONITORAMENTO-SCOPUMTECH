# Project Summary
Scopum Tech is an advanced Smart Video Monitoring System aimed at enhancing security and operational efficiency across commercial and residential environments. The system leverages cutting-edge surveillance technology to deliver real-time monitoring features, including camera registration, live streaming, and comprehensive analytics. Key functionalities encompass facial recognition, sector management, visitor pattern reporting, and AI-driven object detection, ensuring thorough oversight and safety.

# Project Module Description
- **Backend**: Handles data storage, API management, and business logic for monitoring events and zones using FastAPI and SQLAlchemy. It includes a Stream Converter for RTSP/RTMP to HLS conversion, an RTSP snapshot service, and a YOLOv8 detection service for real-time object detection.
- **Frontend**: Provides an intuitive user interface for monitoring, reporting, and managing zones, featuring real-time data visualization, camera management, and AI detection capabilities.
- **Database**: Stores monitoring events, zones, sectors, and facial recognition data, supporting CRUD operations.

# File Description Inventory
- **app/backend/main.py**: Entry point for the backend application.
- **app/backend/lambda_handler.py**: AWS Lambda entry point for routing.
- **app/backend/routers/**: Contains route definitions for handling API requests, including endpoints for stream conversion, RTSP snapshots, and YOLO detection.
- **app/backend/services/stream_converter.py**: Service logic for managing RTSP/RTMP to HLS stream conversion via external media servers.
- **app/backend/services/rtsp_snapshot.py**: Service for capturing JPEG snapshots from RTSP streams using ffmpeg.
- **app/backend/services/yolo_detection.py**: Service for YOLOv8 object detection, processing JPEG frames and returning annotated images with metadata.
- **app/backend/routers/yolo_detection.py**: API endpoints for YOLOv8 detection, including image and JSON responses.
- **app/frontend/src/**: Contains React components and pages for the frontend application, including features for facial recognition, sector management, and AI detection functionality.
- **app/frontend/src/lib/api.ts**: Interfaces and CRUD functions for entities such as camera streams and media server configurations, updated to include YOLO detection API calls.
- **app/frontend/src/pages/Cameras.tsx**: Page for managing cameras with full CRUD operations and media server configuration, now integrated with YOLO detection functionality.
- **app/frontend/src/pages/Monitor.tsx**: Page for monitoring camera streams, updated to handle RTSP URLs and integrate detection features.
- **app/frontend/src/components/StreamPlayer.tsx**: Component updated to support AI detection mode with a toggle for switching between YOLO detection and plain snapshots.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic, httpx, ffmpeg, ultralytics
- **Frontend**: React, TypeScript, Recharts, Tailwind CSS, HLS.js, mpegts.js
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For backend: `pip install -r app/backend/requirements.txt`
   - For frontend: `pnpm install` in `app/frontend`
   
2. **Build and Run**:
   - Backend: Execute `python app/backend/main.py`
   - Frontend: Execute `pnpm run build` then serve the production build.
